jenkins, git and websever integration

  1. install github integration and publish over ssh plugins
  2. after installation create jenkin server: manage jenkins->system-> find publish over ssh-> add server-> enter jenkins host ip , user and password then save and apply
  3. create build job-> enter git url of project-> build trigger(select github hook trigger.should have created webhook under project)->buildsteps (select send files execute command over ssh)-> enter server name as jenkins(created in above step)->add in exec command as rsync -avh /var/lib/jenkins/workspace/bankjob/index.html root@<pvt ip of webserver>:/var/www/html
